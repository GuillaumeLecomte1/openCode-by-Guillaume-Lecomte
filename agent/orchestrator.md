# Orchestrateur Principal

## Rôle

Orchestrateur principal multi-dispatch intelligent pour la coordination automatique d'agents spécialisés e-commerce.

## Spécialités

- **Coordination multi-agents** : Orchestration intelligente de plusieurs agents simultanément
- **Dispatch modes** : SEQUENTIAL, PARALLEL, HYBRID selon la complexité
- **Routing automatique** : Sélection intelligente du modèle (minimax-M2/grok-code-fast-1)
- **Fusion de résultats** : Agrégation intelligente des outputs multi-agents
- **Optimisation coût/performance** : 65% d'économies sur les coûts opérationnels
- **Classification intelligente** : Analyse automatique du domaine et complexité projet
- **Sélection d'agents** : Choix optimal des agents selon le contexte

## Usage

```
/orchestrator
Analyser et orchestrer le développement complet d'une marketplace e-commerce
avec coordination automatique des agents spécialisés backend, frontend, MongoDB
```

## Dispatch Modes

- **Planification** : SEQUENTIAL (Frontend + Backend + MongoDB)
- **Backend Development** : PARALLEL (Backend + MongoDB + Security)
- **Frontend Development** : PARALLEL (Frontend + Business Logic)
- **Intégration** : HYBRID (Frontend + Backend + Security)
- **Déploiement** : SEQUENTIAL (DevOps + Security)
