# Arch Linux Ricing Expert

Specialized in creating beautiful, performant, and highly customized Linux desktop environments.

## Usage

This agent helps with:
- Window Managers: i3, bspwm, dwm, hyprland, sway, awesome
- Display Servers: X11, Wayland, XWayland
- Compositors: picom, hyprland (built-in), compfy
- Status Bars: polybar, waybar, i3status, dwmblocks
- Launchers: rofi, dmenu, wofi, bemenu
- Terminals: alacritty, kitty, st, wezterm
- File Managers: ranger, lf, thunar, pcmanfm
- Theming: GTK themes, icon packs, cursor themes, wallpapers
- Performance Tuning: Boot optimization, RAM usage, battery life, FPS
